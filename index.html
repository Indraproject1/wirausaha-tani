⏸️ MENU PERMAINAN

📖 PANDUAN BERMAIN

🎮 LAYAR SENTUH (HP)

🕹️ D-Pad: Sentuh arah panah untuk menggerakkan petani.

🟢 Tombol A: Interaksi Cangkul / Siram Air ke petak tanah.

🔵 Tombol B: Tanam Benih / Panen tanaman matang / Interaksi bangunan (Toko, Silo, Sumur, Ekspor).

⌨️ KEYBOARD (PC/LAPTOP)

🚶‍♂️ Gerak Petani: Tombol Panah atau W / A / S / D.

🟢 Tombol A: Tekan tombol J pada keyboard.

🔵 Tombol B: Tekan tombol K pada keyboard.

🌱 CARA BERCOCOK TANAM

1. Dekati petak tanah rumput, lalu tekan A (Cangkul).

2. Tekan B (Tanam) pada tanah gembur, lalu pilih benih di tas.

3. Tekan A (Siram) agar tanah basah. Tanaman hanya tumbuh di tanah basah (cokelat tua)!

4. Saat matang (ada buah melayang), tekan B (Panen) untuk memanen (+XP).

🏬 AREA & FITUR BANGUNAN

🏪 Toko Desa (Kiri Bawah): Beli benih padi, jagung, tomat, buah naga.

🌾 Lumbung Silo (Kiri Atas): Simpan hasil panen. Kapasitas terbatas, bisa di-upgrade!

💧 Sumur (Kanan Bawah): Bisa dibangun sumur otomatis (Lvl 3) untuk mempercepat tumbuh 20%.

🚢 Toko Ekspor (Kanan Atas): Dibuka Lvl 4. Harga jual premium 1.5x lipat!

🌟 PROGRESS PERMAINAN

Kumpulkan XP untuk naik Level! Level baru membuka benih tanaman mahal yang lebih menguntungkan: Jagung (Lvl 2), Tomat (Lvl 3), Buah Naga (Lvl 4).
